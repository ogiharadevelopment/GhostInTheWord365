GhostInTheWord - Support

About GhostInTheWord

GhostInTheWord is a Microsoft Word add-in that helps you save and quickly apply text formatting. Save your favorite formats with keyboard shortcuts and apply them instantly to any selected text.

Key Features

  • Save text formatting with a single keystroke
  • Apply saved formats instantly using keyboard shortcuts
  • Continuous mode for automatic format application
  • Bilingual support (English and Japanese)

Free vs Premium

Free Version

  • Save up to 3 formats
  • Basic formatting features
  • All core functionality

Premium Version

  • Unlimited format storage
  • Advanced formatting options
  • Priority support
  • Future feature updates

Quick Start

  1. Select text with the formatting you want to save
  2. Hover over the SAVE area and press any key
  3. To apply: First select text, then hover over the LOAD area, and press the same key you used to save the format
  4. When continuous mode is ON, the format saved to the displayed key will be continuously applied to dragged text
  5. Use mouse wheel on the Font area to adjust font size

Version Information

Current Version: 1.3.0

Compatibility: Microsoft Word

GhostInTheWord - サポート

GhostInTheWordについて

GhostInTheWordは、Microsoft Wordの書式を保存し、素早く適用できるアドインです。 お気に入りの書式をキーボードショートカットで保存し、選択したテキストに瞬時に適用できます。

主な機能

  • テキストの書式を1つのキーで保存
  • 保存した書式をキーボードショートカットで瞬時に適用
  • 連続モードで自動的に書式を適用
  • バイリンガルサポート（日本語・英語）

無料版とプレミアム版

無料版

  • 最大3個の書式を保存
  • 基本的な書式機能
  • すべてのコア機能

プレミアム版

  • 無制限の書式保存
  • 高度な書式オプション
  • 優先サポート
  • 今後の機能アップデート

クイックスタート

  1. 保存したい書式が適用されているテキストを選択
  2. SAVEエリアにマウスカーソルを合わせて任意のキーを押す
  3. 適用時：まずテキストを選択してから、LOADエリアにマウスカーソルを合わせて、SAVEした書式と同じキーを押す
  4. 連続モードがONのときは、表示されているキーに保存された書式を、ドラッグした文章に連続で当てはめます
  5. フォントエリアでマウスホイールを使用してフォントサイズを調整できます

バージョン情報

現在のバージョン: 1.3.0

互換性: Microsoft Word
